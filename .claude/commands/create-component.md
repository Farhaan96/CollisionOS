# Create React Component

Generate a new React component for CollisionOS with all necessary files.

## Task
Use the Code Generator agent to create a new component: $ARGUMENTS

The component should include:
1. Main component file with hooks and Material-UI
2. Styles file (CSS-in-JS)
3. Test file with React Testing Library
4. Index file for exports
5. PropTypes or TypeScript definitions
6. Redux integration if needed

## Component Structure
```
src/components/ComponentName/
  ├── index.js
  ├── ComponentName.js
  ├── ComponentName.test.js
  └── styles.js
```

## Requirements
- Follow CollisionOS component patterns
- Include error boundaries
- Add loading states
- Implement responsive design
- Include accessibility attributes
