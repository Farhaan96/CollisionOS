# Fix Bug

Debug and fix an issue in the CollisionOS codebase.

## Task
Use the Orchestrator to coordinate bug fixing for: $ARGUMENTS

The orchestrator will:
1. Use Search Agent to locate relevant code
2. Use Debugger to diagnose the issue
3. Use Code Generator to implement fix
4. Use Code Reviewer to validate changes
5. Use Test Runner to verify fix
6. Update documentation if needed

## Process
1. **Reproduce** - Confirm the bug exists
2. **Diagnose** - Find root cause
3. **Fix** - Implement solution
4. **Test** - Verify fix works
5. **Review** - Ensure no side effects
6. **Document** - Update relevant docs

## Success Criteria
- Bug no longer reproducible
- All tests passing
- No performance regression
- Code review approved
